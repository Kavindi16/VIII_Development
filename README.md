# VIII-Development Task

This assignment's goal is studying about JavaScript project demonstrating unit testing, code coverage using Coveralls and testing and GitHub Action Workflow.

---

## Tools Used

- **Node.js & npm** – JavaScript runtime and package manager  
- **Mocha** – Test framework  
- **Chai** – Assertion library  
- **nyc** – Code coverage tool  
- **Coveralls** – Code coverage reporting service    

---

## Installation

1. Clone the repository:

```bash
git clone <https://github.com/Kavindi16/VIII_Development.git>
cd VIII_Development
```


2. Install all project dependencies:

```bash
npm install
```
3. Running test
```bash
npm test

npm run coverage
```
